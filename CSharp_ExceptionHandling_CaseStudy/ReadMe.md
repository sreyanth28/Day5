# CSharp_ExceptionHandling_CaseStudy

## Problem Statement
Banking system to handle deposit, withdrawal, and balance check with proper exception handling.

## Exception Types Used
- ArgumentException
- FormatException
- InvalidAmountException (Custom)
- InsufficientBalanceException (Custom)

## Business Rules
- Minimum balance ₹1000
- Deposit must be > 0
- Withdrawal cannot exceed balance

## Sample Output

Case 1:
Enter deposit amount: -500  
ErrorEnter amount greater than zero.

Case 2:
Enter withdrawal amount: 6000  
ErrorInsufficient funds.
